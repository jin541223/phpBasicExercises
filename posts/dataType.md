### 数据类型

1. 四种标量
   - boolean
   - integer [八进制 0，十六进制 0x，二进制 0b]
   - float (double)
   - string [单引号，双引号，heredoc，nowdoc]
     注：表达一个单引号，需要再它的前边加一个反斜线转义；表达一个反斜线自身，则用两个反斜线；

2. 三种复合类型
	- array [key 可以是 integer或string，数组和对象不能被用于键名]
	- object 
	- callable

3. 两种特殊类型
  
    - resource
	- null (无类型)
